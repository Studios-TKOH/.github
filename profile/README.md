## Proyectos

### Patreon DRM System
**Categoría:** Content Protection / Fintech  
**Estado:** En desarrollo  

Sistema de gestión de derechos digitales integrado con Patreon, orientado a la validación de membresías en tiempo real y protección de contenido exclusivo.

**Tecnologías:** OAuth 2.0 · Node.js · Render

---

### Ataraxia
**Categoría:** Herramienta de productividad  
**Estado:** Beta  

Ecosistema de gestión del tiempo diseñado para trabajo profundo, con métricas de productividad, gamificación y sincronización entre dispositivos.

**Tecnologías:** React · Redux Saga · Vite PWA · NestJS

---

### Kardex Pro
**Categoría:** Enterprise / Logística  
**Estado:** MVP completado  

Sistema inteligente de gestión de inventarios con predicción de stock y alertas automatizadas.

**Tecnologías:** Next.js · PostgreSQL · Tailwind CSS
